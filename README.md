# Interactive Periodic Table (iykyk)

Minimal interactive periodic table demo.

Try locally:

```bash
open index.html
```

Files:
- `index.html` — site shell
- `styles.css` — styles
- `elements.js` — sample element data (first 20)
- `app.js` — rendering & interactivity

Notes:
- Keyboard: use arrow keys to move between elements, Enter to open details, Esc to close modal.
- Filtering: use the category buttons in the header to filter elements.

Next steps: expand `elements.js` to include all 118 elements with full data.
